# OneTech Automation - Ecommerce AI Chatbot Platform

## Vision
AI chatbot and automation platform for high-growth ecommerce brands. Turn conversations into predictable revenue by automating sales, support, and customer journeys across WhatsApp, Instagram, and Messenger—without adding headcount.

**Target Audience:** Ecommerce brands serious about scaling (€500k-€10M+ annual revenue)

## Design
Modern SaaS landing page with premium positioning for B2B ecommerce automation.

**Color System:**
- `--primary: 262 83% 58%` (vibrant purple) - Primary CTA, accents, trust elements
- `--accent: 280 90% 65%` (bright magenta) - Gradient highlights, hover states
- `--background: 0 0% 100%` (pure white) - Main background
- `--foreground: 222 47% 11%` (deep navy) - Primary text
- `--muted: 210 40% 96%` (light gray) - Section backgrounds, cards
- `--border: 214 32% 91%` (subtle gray) - Component borders

**Typography:**
- Headings: Plus Jakarta Sans (bold, modern, tech-focused)
- Body: Work Sans (clean, professional, readable)

**Style Direction:** Premium SaaS - clean, professional, conversion-focused. Generous whitespace, gradient accents on CTAs, subtle card hover effects. Optimized for ecommerce B2B buyers.

## Features
Complete conversion-optimized landing page for AI chatbot automation platform:

**Core Sections:**
1. **Hero** - Value proposition with trust badge, dual CTAs, social proof stats
2. **Positioning** - Premium hook addressing conversion problems vs traffic problems
3. **Core Value** - Revenue channel transformation messaging
4. **Platform Authority** - WhatsApp, Instagram, Messenger automation capabilities
5. **Use Cases** - 6 ecommerce automation scenarios (cart recovery, support, upsells)
6. **Results** - Performance benefits (conversions, response time, costs, LTV, scalability)
7. **Process** - 3-step implementation system
8. **Testimonials** - Social proof from ecommerce brands
9. **Pricing** - 4-tier structure (Starter, Growth, Pro, Enterprise)
10. **FAQ** - SEO-optimized ecommerce chatbot questions
11. **Contact Form** - Lead capture with phone number (mandatory), Supabase database, email notifications via Resend
12. **Final CTA** - High-ticket conversion close

**Technical Implementation:**
- Next.js 15 (Page Router), React 18, TypeScript
- Tailwind CSS + shadcn/ui components
- Supabase (database + Edge Functions)
- Resend API (email notifications to support@onetechautomation.com)
- Fully responsive, dark mode support
- SEO optimized meta tags
